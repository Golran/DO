# Обход в ширину #
Формулировка задачи лежит в lab1.rtf.

exe файл создаётся тут ...HorseBFS\bin\Debug или тут ...HorseBFS\bin\Release всё зависит от того в какой конфигурации вы собирали проект Release или Debug.

В папке Debug или Release рядом с exe файлом должны лежать in.txt и out.txt.

Соответственно, меняйте in.txt и запускайте прогу, чтобы потестить, а результат запишется в out.txt.
